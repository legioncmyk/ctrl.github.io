🚀 CARA MENJALANKAN:
══════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│  PILIHAN 1: HTML/CSS/JS (PALING MUDAH)                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. Masuk ke folder: html-css-js-version                            │
│  2. Buka file index.html dengan browser                             │
│  3. SELESAI! Website langsung jalan                                 │
│                                                                     │
│  Tidak perlu install apapun!                                        │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│  PILIHAN 2: Next.js (FULL STACK)                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. Install Node.js atau Bun di komputer                            │
│  2. Buka terminal, masuk ke folder: nextjs-version                  │
│  3. Jalankan:                                                       │
│                                                                     │
│     npm install                                                      │
│     npm run dev                                                      │
│                                                                     │
│     ATAU (jika pakai Bun):                                          │
│                                                                     │
│     bun install                                                      │
│     bun run dev                                                      │
│                                                                     │
│  4. Buka browser: http://localhost:3000                             │
└─────────────────────────────────────────────────────────────────────┘
